Deploy Walkthrough

From code to live site in 5 minutes

After CREAO pushes your code to GitHub, follow these steps to go live.

A

Create a GitHub account

Go to github.com/signup — enter email, password, username. Free.

B

Connect GitHub in CREAO

In CREAO: Profile → Settings → Integrations → GitHub → Authorize. One time only.

C

Create a Vercel account

Go to vercel.com → Sign Up → Continue with GitHub. Choose Hobby (free).

D

Import your repo

Vercel Dashboard → Add New → Project → find your repo in the list → click Import.

E

Verify settings (auto-detected)

Framework: Vite · Build command: npm run build · Output: dist

F

Add env vars (if using Supabase)

Click Environment Variables → add VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY. Get them from Supabase → Settings → API.

G

Click Deploy

Wait ~60 seconds. Vercel builds and deploys. You see confetti 🎉 when it's ready.

H

Your free domain is live

Your site is at yourproject.vercel.app. HTTPS, CDN, auto-deploy on every push. Free forever.
